# 🎬 React Movie Search

Простой поисковик фильмов на React + Vite с использованием [OMDb API](http://www.omdbapi.com/).  
Учебный проект для отработки работы с классовыми компонентами, состоянием и запросами к API.

## 🛠 Технологии

- React (классовые компоненты)
- Vite (сборка)
- OMDb API
- Materialize CSS (стилизация)
